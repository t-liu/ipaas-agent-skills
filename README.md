# AI Agent Skills Marketplace

This project serves as a marketplace for AI agent skills designed for various enterprise integration platforms (e.g., MuleSoft, Workato, Boomi, etc.).

## Tech Stack
- **Frontend**: Vue.js 3, TypeScript, Vite, Tailwind CSS
- **Testing**: Vitest, Vue Test Utils
- **Icons & Fonts**: FontAwesome, Inter & JetBrains Mono

## Getting Started

### Prerequisites
- Node.js (v20+ recommended)
- npm (or yarn / pnpm)

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Testing
```bash
npm run test
```

## Deployment
This application is designed to be deployed to **AWS**.
